# Linea Pro Module

## Description
This is a Titanium module wrapping the Linea Pro hardware SDK for iOS. Currently the SDK version used is 1.96.

## Device support

Universal plugin for following devices:

* PPAD 1.0
* MPED-400
* BluePad-500
* BluePad-50
* Infinea TAB mini
* Infinea TAB 4
* Infinea TAB
* Linea Pro 6
* Linea Pro 5
* Linea Pro 4
* PP-60
* iSerial
* Linea-Pro
